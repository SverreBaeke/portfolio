Project STAM

The goal of this project was to rebuild the STAM website, focusing on modern HTML and CSS practices. This was my first project where I applied a component-based approach to styling. Instead of writing all the CSS in one file, I created separate CSS files for each component of the webpage, such as the header, footer, navigation, and main content sections.

What I Learned

In this project, I learned how to work with a component-based approach to CSS, which was a significant milestone in my development journey. Instead of writing all styles in one single CSS file, I organized my CSS into separate files for each individual component of the webpage. This modular approach made the code much easier to manage, maintain, and scale.

It also helped me understand the importance of separation of concerns, keeping the structure and styling of the webpage organized and distinct. By working this way, I was able to avoid style conflicts and make the components reusable across the project.

Additionally, I gained practical experience with responsive design techniques, ensuring that the webpage adapts smoothly to different screen sizes and devices.

Overall, this project strengthened my foundational CSS skills and taught me best practices that will be essential for building larger and more complex front-end projects in the future.
