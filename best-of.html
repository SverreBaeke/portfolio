Project Best of 2024

This project focused on building a dynamic web application that interacts with an API provided by my school. The main goal was to display data efficiently while improving user experience through pagination and filtering. It was my first time handling asynchronous data fetching and dynamically updating the UI based on the received content.

What I Learned

This project was my first experience implementing pagination, which taught me how to efficiently manage large datasets by loading data in smaller, manageable chunks. Fetching data asynchronously using async/await helped me understand how to handle asynchronous operations cleanly and effectively, especially when working with APIs.

Working with an API hosted by the school gave me practical insight into integrating external data sources and managing real-world data flows. I also learned how to dynamically render content based on the fetched data, making the user interface more interactive and responsive.

In addition, I applied small filtering techniques to allow users to refine the displayed data easily, which improved the overall usability and user experience of the application.

Together, these skills expanded my ability to build dynamic, user-friendly web applications that efficiently handle and present data..
